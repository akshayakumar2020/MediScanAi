# 🩺 MediScan AI

An AI-powered healthcare platform that enables patients to book appointments, upload medical reports, receive AI-assisted report analysis, and connect with doctors through a secure web application.

## 🚀 Live Demo

🌐 **Website:** https://healthcarereport.vercel.app/

📂 **GitHub Repository:** https://github.com/akshayakumar2020/MediScanAi

---

# ✨ Features

### 👤 Patient
- Register & Login securely
- Book doctor appointments
- Upload medical reports (PDF/Image)
- AI-powered report analysis
- View appointment history
- Receive medication reminders
- Google Calendar integration

### 👨‍⚕️ Doctor
- Manage appointments
- Review uploaded reports
- Add diagnosis & prescriptions
- AI-generated patient-friendly summaries
- Manage leave schedule

### 🛠️ Admin
- Manage users
- Approve or reject doctors
- View analytics
- Monitor system health

---

# 🤖 AI Features

- Medical report analysis using Google Gemini
- OCR extraction from scanned reports
- AI-generated medical summaries
- AI medical chatbot assistance
- Patient-friendly explanation of doctor's notes

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/akshayakumar2020/MediScanAi.git
cd MediScanAi
```

## Backend

```bash
cd backend
mvn clean install
mvn spring-boot:run
```

Backend runs on:

```
http://localhost:8080
```

## Frontend

```bash
cd frontend
npm install
npm run dev
```

Frontend runs on:

```
http://localhost:5173
```
